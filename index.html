Tópico 1

Conjuntos Numéricos

Teoria Essencial

Os conjuntos numéricos agrupam números com características semelhantes. A base de tudo começa com os Naturais e se expande até os Reais.

  • Naturais (N): {0, 1, 2, 3, ...}
  • Inteiros (Z): {..., -2, -1, 0, 1, 2, ...}
  • Racionais (Q): Frações e dízimas periódicas. Ex: 1/2, 0.333...
  • Irracionais (I): Dízimas não periódicas. Ex: π, √2
  • Reais (R): A união de todos os anteriores (Q ∪ I).

Definição

Um conjunto é uma coleção de elementos bem definidos. A relação de pertinência (∈) liga um elemento a um conjunto. A relação de inclusão (⊂) liga dois conjuntos.

Exemplo Resolvido

Problema: Identifique a qual conjunto pertence o número -5 e o número √3.

1

-5 é um número negativo sem parte decimal. Logo, pertence aos Inteiros (Z) e também aos Racionais (Q) e Reais (R).

2

√3 não possui raiz exata, resultando em uma dízima infinita e não periódica (1.73205...). Logo, pertence aos Irracionais (I) e Reais (R).

Atenção na Prova

Erro Comum

Confundir dízimas periódicas (ex: 0,333...) com números Irracionais. Dízimas periódicas podem ser escritas como fração (1/3), portanto são Racionais (Q).

Macete

Lembre da boneca russa (Matrioska): N está dentro de Z, que está dentro de Q. O I fica separado. Juntando Q e I, temos a boneca maior: R.

Tópico 2

Intervalos Reais

Teoria Essencial

Intervalos são subconjuntos dos números Reais (R) que representam todos os números entre dois extremos.

Representações

Aberto: ]a, b[ ou (a, b) -> Não inclui os extremos.
Fechado: [a, b] -> Inclui os extremos.
Misto: [a, b[ -> Inclui 'a', mas não 'b'.

Tópico 3

Álgebra e Fatoração

Teoria Essencial

Fatorar é transformar uma soma ou subtração em um produto. Produtos notáveis são multiplicações frequentes que possuem regras fixas.

Produtos Notáveis

Quadrado da soma: (a+b)² = a² + 2ab + b²
Quadrado da diferença: (a-b)² = a² - 2ab + b²
Produto da soma pela diferença: (a+b)(a-b) = a² - b²

Tópico 4

Relação de Ordem e Reta Real

Teoria Essencial

A reta real é uma representação geométrica do conjunto dos números reais (R). Cada ponto na reta corresponde a um único número real, e cada número real corresponde a um único ponto.

Relação de Ordem

Dados dois números reais a e b, apenas uma das afirmações é verdadeira:
a < b (a é menor que b, fica à esquerda na reta)
a = b (a é igual a b, mesmo ponto)
a > b (a é maior que b, fica à direita na reta)

Exemplo Resolvido

Problema: Ordene os números: -2, 0, 1.5, -1/2, √2.

1

Converta todos para decimal para facilitar a comparação: -1/2 = -0.5 e √2 ≈ 1.41.

2

Ordene do menor (mais à esquerda) para o maior (mais à direita): -2 < -0.5 < 0 < 1.41 < 1.5.

3

Resposta final: -2 < -1/2 < 0 < √2 < 1.5.

Tópico 5

Potenciação e Radiciação

Teoria Essencial

Potenciação é uma multiplicação de fatores iguais. Radiciação é a operação inversa da potenciação.

Propriedades da Potenciação

  • am · an = am+n (Multiplicação de mesma base)
  • am / an = am-n (Divisão de mesma base)
  • (am)n = am·n (Potência de potência)
  • a-n = 1 / an (Expoente negativo inverte a base)
  • am/n = n√am (Expoente fracionário vira raiz)
