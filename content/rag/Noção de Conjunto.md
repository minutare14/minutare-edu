### **📘 Noção de conjunto**

#### **🧠 Resumo rápido (2–4 linhas)**

Um conjunto é simplesmente uma coleção de objetos, números ou ideias que compartilham alguma característica. Para a prova, o mais importante é saber dizer se um elemento está ou não dentro desse conjunto usando os símbolos de pertinência (\\in e \\notin).

#### **📖 Explicação**

Pense em um conjunto como uma **caixa organizadora**. Você pode ter uma caixa para "vogais", uma para "números pares" ou uma caixa vazia. Os itens dentro da caixa são chamados de **elementos**. Na matemática, geralmente damos nomes aos conjuntos usando letras maiúsculas (A, B, C) e colocamos os elementos entre chaves \\{ \\}. A relação básica aqui é entre o elemento e a caixa: ou ele está lá dentro (pertence) ou não está (não pertence).

#### **📌 Definições importantes**

* **Conjunto:** Coleção de elementos (ex: A \= \\{1, 2, 3\\}).  
* **Elemento:** Cada item individual do conjunto.  
* **Pertence (\\in):** Indica que um elemento faz parte do conjunto. Ex: 2 \\in A.  
* **Não pertence (\\notin):** Indica que o elemento não está no conjunto. Ex: 5 \\notin A.  
* **Representações:** \* Por extensão: listando tudo. V \= \\{a, e, i, o, u\\}  
  * Por propriedade: V \= \\{x \\mid x \\text{ é vogal do alfabeto}\\}

#### **🔍 Exemplos resolvidos**

**Exemplo 1:** Dado o conjunto B \= \\{2, 4, 6, 8\\}, classifique como verdadeiro ou falso:

1.   
2. 

*Passo a passo:*

1. Olhamos para o conjunto B. O número 4 está lá? Sim. Logo, 4 \\in B é **Verdadeiro**.  
2. O número 7 está lá? Não. O correto seria 7 \\notin B. Logo, a afirmação 7 \\in B é **Falsa**.

**Exemplo 2:** Escreva o conjunto C \= \\{x \\mid x \\text{ é número natural ímpar menor que 6}\\} listando seus elementos. *Passo a passo:*

1. Números naturais são inteiros não negativos: 0, 1, 2, 3, 4, 5, 6 \\dots  
2. Precisa ser ímpar: 1, 3, 5, 7 \\dots  
3. Precisa ser menor que 6: 1, 3, 5\. *Resposta:* C \= \\{1, 3, 5\\}

#### **⚠️ Erros comuns**

* **Confundir os símbolos:** Usar \\in (pertence) para relacionar um conjunto com outro conjunto. O símbolo \\in é **exclusivo** para ligar um **elemento** a um **conjunto**.  
* **Esquecer do zero:** Ao trabalhar com números naturais pares menores que 5, muitos esquecem que o 0 entra: \\{0, 2, 4\\}.

#### **💡 Macetes de prova**

* A letra **E** arredondada (\\in) significa **E**stá lá dentro (pertence).  
* Sempre leia "|" como "tal que" ou "de modo que". Isso ajuda a traduzir o "matematiquês" para o português.

#### **🧪 Exercícios**

1. Sendo A \= \\{0, 2, 4, 6\\} e B \= \\{1, 3, 5\\}, assinale a alternativa correta: a) 2 \\in B b) 0 \\notin A c) 5 \\in B d) 6 \\in B  
2. Escreva listando os elementos o conjunto P \= \\{x \\mid x \\text{ é a letra da palavra MATEMATICA}\\}.  
3. Verdadeiro ou falso: Se M \= \\{x \\mid x \\text{ é mês do ano começando com J}\\}, então \\text{Janeiro} \\in M.

**Gabarito:**

1. Letra **c**. (2 está em A, 0 está em A, 6 está em A).  
2. P \= \\{M, A, T, E, I, C\\}. (Lembre-se: em conjuntos, não precisamos repetir elementos idênticos).  
3. **Verdadeiro**. Os meses são Janeiro, Junho e Julho.

#### **🚀 Revisão rápida**

* \[ \] Conjunto \= caixa com coisas (letras maiúsculas).  
* \[ \] Elementos \= itens da caixa (entre chaves).  
* \[ \] \\in \= Elemento PERTENCE ao conjunto.  
* \[ \] Elementos repetidos não contam duas vezes.
