Símbolos de Conjunto

Ex: 3 ∈ ℕ  ·  π ∈ 𝕀
∉
Não pertence
Ex: −2 ∉ ℕ
⊂
Está contido — liga conjunto a conjunto
Ex: ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ
∅
Conjunto vazio — ∅ ⊂ a todo conjunto
∪ ∩
∪ = União (junta tudo)  ·  ∩ = Interseção (apenas o que repete)

Produtos Notáveis
²
(a + b)² = a² + 2ab + b²
Não mate o 2ab!
²
(a − b)² = a² − 2ab + b²
±
(a+b)(a−b) = a² − b²
Soma × diferença = diferença de quadrados
³
(a+b)³ = a³ + 3a²b + 3ab² + b³
³
(a−b)³ = a³ − 3a²b + 3ab² − b³
Fatoração — Fluxograma
①
Fator em evidência: algo se repete?
ax + ay = a(x + y)
②
Diferença de quadrados: dois termos com −
a² − b² = (a+b)(a−b)
③
Trinômio quadrado perfeito: 3 termos
a² ± 2ab + b² = (a ± b)²
④
Agrupamento: 4 termos, evidência em etapas

⚠ Só cancela termos em frações se estiverem multiplicando (fatorados)

Propriedades da Álgebra
🔄
Comutativa: a+b = b+a  ·  a·b = b·a
Não vale para − e ÷
( )
Associativa: (a+b)+c = a+(b+c)
Troca os parênteses, não a ordem
🚿
Distributiva (chuveirinho):
a(b+c) = ab + ac
O carteiro visita TODOS os termos de dentro
0·1
Neutros: x + 0 = x  ·  x · 1 = x
Relação de Ordem
🐊
O jacaré come o MAIOR: 5 > 2, −1 > −8
−
Negativos: −8 < −2 (mais à esquerda = menor)
Imagine o termômetro — dever 8 é pior que dever 2
⚠
Mult./div. por negativo → inverte o sinal!
se a < b e c < 0, então ac > bc
×
Frações — multiplicação cruzada:
3/4 vs 5/7 → 7×3=21, 4×5=20 → 3/4 > 5/7
