Tópico 1

Conjuntos Numéricos

Teoria Essencial

Os conjuntos numéricos agrupam números com características semelhantes. A hierarquia vai dos Naturais até os Reais.

  • Naturais (ℕ): {0, 1, 2, 3, …} — sem negativos nem frações
  • Inteiros (ℤ): {…, −2, −1, 0, 1, 2, …} — inclui negativos
  • Racionais (ℚ): p/q com q≠0. Frações e dízimas periódicas. Ex: 1/2, 0,333…
  • Irracionais (𝕀): Dízimas não periódicas, sem fração. Ex: π, √2, e
  • Reais (ℝ): União de todos: ℝ = ℚ ∪ 𝕀

📌 Pertinência vs. Inclusão

Pertinência (∈): liga um elemento ao conjunto. Ex: 3 ∈ ℕ
Inclusão (⊂): liga dois conjuntos. Ex: ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ

🎨 Diagrama dos Conjuntos — Matrioska

ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ  ·  𝕀 ⊂ ℝ  ·  ℚ ∩ 𝕀 = ∅ (sem elementos em comum)

Exemplo Resolvido

Problema: Classifique nos conjuntos: −5, √3, 0,777…, 2/3.

1
−5: inteiro negativo → pertence a ℤ, ℚ e ℝ. (Não a ℕ.)
2
√3 ≈ 1,732…: dízima infinita não periódica → pertence a 𝕀 e ℝ apenas.
3
0,777… = 7/9: dízima periódica, tem fração equivalente → pertence a ℚ e ℝ.
4
2/3: fração com denominador inteiro → pertence a ℚ e ℝ.
Atenção na Prova

❌ Erro Comum

Confundir dízimas periódicas (0,333…) com irracionais. Dízimas periódicas são racionais — têm fração equivalente. Só são irracionais as dízimas sem período que se repete.

💡 Macete — Boneca Russa

ℕ cabe dentro de ℤ, que cabe dentro de ℚ. O 𝕀 fica separado de ℚ, mas ambos cabem dentro de ℝ. Se pertence a ℕ, automaticamente pertence a todos os maiores.

Tópico 2

Intervalos Reais

Teoria Essencial

Um intervalo é um subconjunto contínuo de ℝ — representa todos os reais entre dois extremos a e b.

📌 Tipos de Intervalo

Fechado [a, b]: inclui ambos os extremos.  a ≤ x ≤ b
Aberto ]a, b[: exclui ambos os extremos.  a < x < b
Misto [a, b[: inclui a, exclui b.  a ≤ x < b
Misto ]a, b]: exclui a, inclui b.  a < x ≤ b
Semirretas: [a, +∞[ ou ]−∞, b] — um extremo finito

💡 Regra do Colchete

[ ] = fechado → inclui o extremo (ponto ●)
] [ = aberto → exclui o extremo (ponto ○)

🎨 Visualização dos Intervalos na Reta Real

O colchete indica se o extremo está incluído ou excluído do conjunto

Exemplos Resolvidos

Ex 1: Represente {x ∈ ℝ | −1 ≤ x < 4}.

1
≤ no extremo esquerdo → inclui −1 → colchete [
2
< no extremo direito → exclui 4 → colchete [
3
Resposta: [−1, 4[

Ex 2: O intervalo ]2, 7] corresponde a qual desigualdade?

1
] no início → exclui 2 → sinal <
2
] no final → inclui 7 → sinal ≤
3
Resposta: 2 < x ≤ 7
Atenção na Prova

❌ Infinito nunca é fechado

O ∞ nunca usa colchete fechado. [a, +∞] está errado — o correto é sempre [a, +∞[.

💡 Macete rápido

≤ ou ≥ → [ (fecha, inclui)  |  < ou > → ] (abre, exclui)

Tópico 3

Álgebra e Fatoração

Produtos Notáveis

Produtos notáveis são multiplicações algébricas tão frequentes que vale memorizar o resultado — economiza tempo na prova.

📌 Fórmulas Fundamentais

Quadrado da Soma(a + b)² = a² + 2ab + b²
Quadrado da Diferença(a − b)² = a² − 2ab + b²
Produto da Soma pela Diferença(a+b)(a−b) = a² − b²
Cubo da Soma(a+b)³ = a³ + 3a²b + 3ab² + b³
Cubo da Diferença(a−b)³ = a³ − 3a²b + 3ab² − b³

🎨 Por que (a+b)² = a² + 2ab + b²?

O quadrado de lado (a+b) tem área = a² + ab + ab + b²

Técnicas de Fatoração

Fatorar é o inverso: transformar uma expressão em um produto de fatores.

📌 5 Métodos (em ordem de tentativa)

1. Fator em evidência: ax + ay = a(x + y)

2. Agrupamento: ab + ac + db + dc = (a+d)(b+c)

3. Diferença de quadrados: a² − b² = (a+b)(a−b)

4. Trinômio quadrado perfeito: a² ± 2ab + b² = (a ± b)²

5. Cubo perfeito: a³ ± 3a²b + 3ab² ± b³ = (a ± b)³

Exemplos Resolvidos

Ex 1 — Fator em Evidência: Fatore 6x² + 9x.

1
MDC(6,9)=3 e MDC(x²,x)=x → fator = 3x
2
6x² + 9x = 3x(2x + 3) ✓

Ex 2 — Diferença de Quadrados: Fatore 4x² − 25.

1
4x² = (2x)² e 25 = 5²
2
4x² − 25 = (2x+5)(2x−5) ✓

Ex 3 — Trinômio Quadrado Perfeito: Fatore x² + 6x + 9.

1
√x²=x, √9=3, e 2·x·3=6x ✓ → é TQP
2
x² + 6x + 9 = (x+3)² ✓
Atenção na Prova

❌ O erro clássico do quadrado da soma

(a+b)² ≠ a² + b² ← ERRADO!
O termo 2ab não pode ser esquecido.
(a+b)² = a² + 2ab + b² ← correto

💡 Macete — Diferença de Quadrados

Viu subtração entre dois quadrados perfeitos? Fatore imediatamente:
x² − 16 = (x+4)(x−4)  |  9y² − 1 = (3y+1)(3y−1)

Tópico 4

Relação de Ordem e Reta Real

Teoria Essencial

A reta real é a representação geométrica de ℝ — cada ponto corresponde a exatamente um número real.

📌 Tricotomia (Relação de Ordem)

Para quaisquer a, b ∈ ℝ, exatamente uma das situações é verdadeira:
a < b → a fica à esquerda de b na reta
a = b → mesmo ponto
a > b → a fica à direita de b na reta

📌 Propriedades da Ordem

Transitividade: a < b e b < c → a < c
Adição: a < b → a+c < b+c (para qualquer c)
× positivo: a < b e c > 0 → ac < bc
× negativo: a < b e c < 0 → ac > bc ← inverte o sinal!

🎨 A Reta Real

Quanto mais à direita, maior o número  |  quanto mais à esquerda, menor

Exemplo Resolvido

Problema: Ordene do menor para o maior: −2, 0, 1,5, −1/2, √2.

1
Converta para decimal: −1/2 = −0,5 e √2 ≈ 1,41.
2
Compare: −2 < −0,5 < 0 < 1,41 < 1,5
3
Resposta: −2 < −1/2 < 0 < √2 < 1,5 ✓
Atenção na Prova

❌ Multiplicar por número negativo

Ao multiplicar ou dividir uma desigualdade por número negativo, o sinal inverte!
−2x < 6 → divide por −2 → x > −3

💡 Raízes úteis para comparar

√2 ≈ 1,41  |  √3 ≈ 1,73  |  √5 ≈ 2,24  |  √10 ≈ 3,16

Tópico 5

Potenciação e Radiciação

Potenciação — Propriedades

aⁿ = a × a × … × a (n vezes). a = base, n = expoente.

📌 Propriedades

Mesma base — produtoaᵐ · aⁿ = aᵐ⁺ⁿ
Mesma base — quocienteaᵐ ÷ aⁿ = aᵐ⁻ⁿ
Potência de potência(aᵐ)ⁿ = aᵐ·ⁿ
Expoente zeroa⁰ = 1  (a ≠ 0)
Expoente negativoa⁻ⁿ = 1/aⁿ
Expoente fracionárioaᵐ/ⁿ = ⁿ√(aᵐ)
Radiciação — Propriedades

Radiciação é a inversa da potenciação: ⁿ√a = b ⟺ bⁿ = a.

📌 Propriedades da Radiciação

ⁿ√(a·b) = ⁿ√a · ⁿ√b — produto de radicandos
ⁿ√(a/b) = ⁿ√a / ⁿ√b — quociente de radicandos
ᵐ√(ⁿ√a) = ᵐ·ⁿ√a — raiz de raiz: multiplica índices
²√(a²) = |a| — resultado sempre ≥ 0
²√a só existe em ℝ se a ≥ 0

💡 Simplificação de Raízes

Para simplificar √72:
√72 = √(36·2) = √36·√2 = 6√2
Encontre sempre o maior quadrado perfeito que divide o radicando.

Exemplos Resolvidos

Ex 1: Simplifique (2³)² ÷ 2⁴

1
(2³)² = 2^(3·2) = 2⁶ = 64
2
2⁶ ÷ 2⁴ = 2^(6−4) = 2² = 4 ✓

Ex 2: Calcule 3⁻²

1
3⁻² = 1/3² = 1/9 ✓

Ex 3: Simplifique 8^(2/3)

1
8^(2/3) = (³√8)² = 2² = 4 ✓

Ex 4: Simplifique √48

1
48 = 16 · 3 (16 é quadrado perfeito)
2
√48 = √16 · √3 = 4√3 ✓
Atenção na Prova

❌ Bases diferentes não se unem

2³ · 3² ≠ 6⁵ ← ERRADO!
Só pode juntar potências com a mesma base.

❌ Raiz de soma ≠ soma de raízes

√(a² + b²) ≠ a + b ← ERRADO!
A raiz distribui sobre produto, não sobre soma.
√(9+16) = √25 = 5 ≠ 3+4 = 7

√4= 2√25= 5
