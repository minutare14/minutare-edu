Tópico 1

Conjuntos Numéricos

Teoria Essencial

Os conjuntos numéricos agrupam números com características semelhantes. A hierarquia vai dos Naturais (mais simples) até os Reais (mais completos).

  • Naturais (ℕ): {0, 1, 2, 3, ...} — não têm negativos nem frações
  • Inteiros (ℤ): {..., −2, −1, 0, 1, 2, ...} — incluem negativos
  • Racionais (ℚ): p/q com q≠0 — frações e dízimas periódicas. Ex: 1/2, 0,333…
  • Irracionais (𝕀): Dízimas não periódicas, não viram fração. Ex: π, √2, e
  • Reais (ℝ): Todos os anteriores: ℝ = ℚ ∪ 𝕀

📌 Pertinência vs. Inclusão

Pertinência (∈): liga um elemento a um conjunto. Ex: 3 ∈ ℕ
Inclusão (⊂): liga dois conjuntos. Ex: ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ

🎨 Diagrama dos Conjuntos (Matrioska)

ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ  |  𝕀 ⊂ ℝ  |  ℚ ∩ 𝕀 = ∅

Exemplo Resolvido

Problema: Classifique cada número nos conjuntos a que pertence: −5, √3, 0,777…, 2/3.

1
−5: inteiro negativo → pertence a ℤ, ℚ e ℝ (mas não a ℕ).
2
√3 ≈ 1,7320508…: dízima infinita não periódica → pertence a 𝕀 e ℝ apenas.
3
0,777… = 7/9: dízima periódica, pode virar fração → pertence a ℚ e ℝ.
4
2/3 = 0,666…: fração com denominador inteiro → pertence a ℚ e ℝ.
Atenção na Prova

❌ Erro Comum

Confundir dízimas periódicas (0,333…) com irracionais. Dízimas periódicas são racionais — têm fração equivalente. Só são irracionais as dízimas sem período que se repete.

💡 Macete — Boneca Russa (Matrioska)

ℕ fica dentro de ℤ, que fica dentro de ℚ. O 𝕀 fica separado de ℚ, mas ambos cabem dentro de ℝ. Se pertence a ℕ, automaticamente pertence a todos os maiores.

Tópico 2

Intervalos Reais

Teoria Essencial

Um intervalo é um subconjunto contínuo de ℝ — representa todos os reais entre dois extremos a e b.

📌 Tipos de Intervalo

Fechado [a, b]: inclui os extremos. a ≤ x ≤ b
Aberto ]a, b[ ou (a, b): exclui os extremos. a < x < b
Misto [a, b[: inclui a, exclui b. a ≤ x < b
Misto ]a, b]: exclui a, inclui b. a < x ≤ b
Semirretas: [a, +∞[ ou ]−∞, b] — só um extremo finito

💡 Regra do Colchete

[ ] = colchete fechado → inclui o extremo (ponto preenchido ●)
] [ = colchete aberto → exclui o extremo (ponto vazio ○)

🎨 Visualização dos Intervalos

Ponto preenchido (●) = inclui o extremo  |  Ponto vazio (○) = exclui o extremo

Exemplos Resolvidos

Ex 1: Represente o conjunto dos x tais que −1 ≤ x < 4.

1
O sinal ≤ no extremo esquerdo significa inclui −1 → colchete [
2
O sinal < no extremo direito significa exclui 4 → colchete [
3
Resposta: [−1, 4[

Ex 2: O intervalo ]2, 7] corresponde a qual desigualdade?

1
] no início → exclui 2 → sinal <
2
] no final → inclui 7 → sinal ≤
3
Resposta: 2 < x ≤ 7
Atenção na Prova

❌ Erro Comum

O infinito (+∞ ou −∞) nunca é incluído — sempre use colchete aberto ao lado do infinito. Escrever [a, +∞] está errado!

❌ Outro Erro Frequente

Confundir a notação americana (a, b) com intervalo aberto. Em provas brasileiras, use ]a, b[ para aberto e [a, b] para fechado.

💡 Macete Rápido

Leia o sinal de desigualdade e escolha o colchete:
≤ ou ≥ → [ (fechado, inclui)
< ou > → ] (aberto, exclui)

Tópico 3

Álgebra e Fatoração

Produtos Notáveis

Produtos notáveis são multiplicações algébricas tão comuns que vale a pena memorizar o resultado direto — economiza tempo na prova.

📌 Fórmulas Fundamentais

Quadrado da Soma (a + b)² = a² + 2ab + b²
Quadrado da Diferença (a − b)² = a² − 2ab + b²
Produto da Soma pela Diferença (a + b)(a − b) = a² − b²
Cubo da Soma (a + b)³ = a³ + 3a²b + 3ab² + b³
Cubo da Diferença (a − b)³ = a³ − 3a²b + 3ab² − b³

🎨 Por que (a+b)² = a² + 2ab + b²?

(a+b)² = área do quadrado grande = a² + ab + ab + b²

Técnicas de Fatoração

Fatorar é o inverso: transformar uma expressão em um produto de fatores.

📌 Métodos de Fatoração (em ordem de tentativa)

1. Fator comum em evidência: ax + ay = a(x + y)

2. Agrupamento: ab + ac + db + dc = a(b+c) + d(b+c) = (a+d)(b+c)

3. Diferença de quadrados: a² − b² = (a+b)(a−b)

4. Trinômio quadrado perfeito: a² ± 2ab + b² = (a ± b)²

5. Cubo perfeito: a³ ± 3a²b + 3ab² ± b³ = (a ± b)³

Exemplos Resolvidos

Ex 1 — Fator em Evidência: Fatore 6x² + 9x.

1
MDC(6, 9) = 3 e MDC(x², x) = x → fator comum = 3x
2
6x² + 9x = 3x(2x + 3) ✓

Ex 2 — Diferença de Quadrados: Fatore 4x² − 25.

1
Identifique: 4x² = (2x)² e 25 = 5²
2
Use a² − b² = (a+b)(a−b)
3
4x² − 25 = (2x + 5)(2x − 5) ✓

Ex 3 — Trinômio Quadrado Perfeito: Fatore x² + 6x + 9.

1
Verifique: √(x²) = x, √9 = 3, e 2·x·3 = 6x ✓
2
x² + 6x + 9 = (x + 3)² ✓
Atenção na Prova

❌ Erro Clássico no Quadrado da Soma

(a + b)² ≠ a² + b² ← ERRADO!
O termo do meio (2ab) não pode ser esquecido.
(a + b)² = a² + 2ab + b² ← certo

💡 Macete — Diferença de Quadrados

Viu subtração entre dois quadrados perfeitos? Fatore imediatamente:
x² − 16 = (x+4)(x−4)
9y² − 1 = (3y+1)(3y−1)

Tópico 4

Relação de Ordem e Reta Real

Teoria Essencial

A reta real é a representação geométrica de ℝ — cada ponto corresponde exatamente a um número real, e vice-versa.

📌 Relação de Ordem (Tricotomia)

Para quaisquer a, b ∈ ℝ, exatamente uma das afirmações é verdadeira:
a < b → a fica à esquerda de b na reta
a = b → mesmo ponto
a > b → a fica à direita de b na reta

📌 Propriedades da Ordem

Transitividade: a < b e b < c → a < c
Adição: a < b → a + c < b + c (para qualquer c)
Multiplicação: a < b e c > 0 → ac < bc
Inversão: a < b e c < 0 → ac > bc ← atenção!

🎨 A Reta Real

Quanto mais à direita, maior o número. Quanto mais à esquerda, menor.

Exemplo Resolvido

Problema: Ordene do menor para o maior: −2, 0, 1,5, −1/2, √2.

1
Converta tudo para decimal: −1/2 = −0,5 e √2 ≈ 1,41.
2
Compare: −2 < −0,5 < 0 < 1,41 < 1,5
3
Resposta: −2 < −1/2 < 0 < √2 < 1,5
Atenção na Prova

❌ Erro Gravíssimo — Multiplicar por Negativo

Ao multiplicar ou dividir uma desigualdade por número negativo, o sinal inverte!
−2x < 6 → x > −3 (dividiu por −2, inverteu)

💡 Dica para Comparar Números

Ao comparar frações, raízes e decimais, converta tudo para decimal primeiro. Use calculadora mental para √2 ≈ 1,41, √3 ≈ 1,73, √5 ≈ 2,23.

Tópico 5

Potenciação e Radiciação

Potenciação — Teoria e Propriedades

Potenciação: aⁿ = a × a × a … (n vezes). Chamamos a de base e n de expoente.

📌 Propriedades da Potenciação

Produto de mesma base aᵐ · aⁿ = aᵐ⁺ⁿ
Quociente de mesma base aᵐ ÷ aⁿ = aᵐ⁻ⁿ
Potência de potência (aᵐ)ⁿ = aᵐ·ⁿ
Expoente zero a⁰ = 1 (a ≠ 0)
Expoente negativo a⁻ⁿ = 1/aⁿ
Expoente fracionário aᵐ/ⁿ = ⁿ√(aᵐ)
Radiciação — Teoria e Propriedades

Radiciação é a operação inversa da potenciação: ⁿ√a = b se bⁿ = a.

📌 Propriedades da Radiciação

ⁿ√(a · b) = ⁿ√a · ⁿ√b (produto de radicandos)
ⁿ√(a / b) = ⁿ√a / ⁿ√b (quociente de radicandos)
ᵐ√(ⁿ√a) = ᵐ·ⁿ√a (raiz de raiz: multiplica índices)
√(a²) = |a| (resultado sempre positivo ou zero)
²√a só existe em ℝ se a ≥ 0

💡 Simplificação de Raízes

Para simplificar √72:
√72 = √(36 · 2) = √36 · √2 = 6√2
Procure sempre o maior quadrado perfeito que divide o radicando.

Exemplos Resolvidos

Ex 1 — Propriedades de Potência: Simplifique (2³)² ÷ 2⁴

1
(2³)² = 2^(3·2) = 2⁶ = 64
2
2⁶ ÷ 2⁴ = 2^(6−4) = 2² = 4 ✓

Ex 2 — Expoente Negativo: Calcule 3⁻²

1
3⁻² = 1/3² = 1/9 ✓

Ex 3 — Expoente Fracionário: Simplifique 8^(2/3)

1
8^(2/3) = ³√(8²) = ³√64 = 4 ✓
2
Ou: 8^(2/3) = (8^(1/3))² = (³√8)² = 2² = 4 ✓

Ex 4 — Simplificar Raiz: Calcule √48

1
Fatore 48: 48 = 16 · 3 (16 é quadrado perfeito)
2
√48 = √(16 · 3) = √16 · √3 = 4√3 ✓
Atenção na Prova

❌ Erro de Bases Diferentes

2³ · 3² ≠ 6⁵ ← ERRADO!
Só pode juntar potências com a mesma base.

❌ Raiz Quadrada de Soma

√(a² + b²) ≠ a + b ← ERRADO!
A raiz distribui sobre produto, não sobre soma.
√(9 + 16) = √25 = 5 ≠ 3 + 4 = 7

💡 Raízes Importantes para Memorizar

√nValor√nValor
√2≈ 1,41√9= 3
√3≈ 1,73√16= 4
√4= 2√25= 5
√5≈ 2,24√100= 10
