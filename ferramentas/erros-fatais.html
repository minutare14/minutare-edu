Erros Fatais — CTIA03

Os erros mais comuns compilados de todos os módulos. Leia isso antes de entrar na prova.

14
erros documentados diretamente nos materiais do curso, organizados por tópico.

Conjuntos

❌ Errado Noção de Conjunto
ℕ ∈ ℤ

Usar ∈ para relacionar dois conjuntos.

✓ Correto
ℕ ⊂ ℤ

∈ liga elemento → conjunto.  ⊂ liga conjunto → conjunto.

❌ Errado Conjunto Vazio
{∅} = conjunto vazio

Isso é um conjunto com um elemento dentro (o símbolo ∅).

✓ Correto
∅   ou   { }

Conjunto vazio é representado apenas por ∅ ou chaves vazias { }.

❌ Errado Diferença
A − B = B − A

Achar que a diferença é comutativa.

✓ Correto
A − B ≠ B − A (em geral)

A−B são os elementos exclusivos de A. B−A são os exclusivos de B. Resultados diferentes.

Conjuntos Numéricos

❌ Errado Racional vs Irracional
0,777… é irracional

O aluno vê "infinito" e marca irracional.

✓ Correto
0,777… = 7/9 → é RACIONAL (ℚ)

Dízima periódica tem fração equivalente → é racional. Irracional só se for decimal sem padrão que se repete.

❌ Errado Raízes
√16 é irracional

Achar que toda raiz quadrada é irracional.

✓ Correto
√16 = 4 → é NATURAL (ℕ)

Resolva a raiz antes de classificar. √4=2, √9=3, √25=5 são todos naturais.

Intervalos Reais

❌ Errado Infinito
[2, +∞]

Fechar o intervalo no infinito com colchete ].

✓ Correto
[2, +∞[  ou  [2, +∞)

Infinito não é um número — nunca pode ser incluído. Sempre aberto.

❌ Errado Bolinha na reta
Bolinha vazia para ≤ ou ≥

Esquecer de pintar a bolinha quando o sinal inclui o extremo.

✓ Correto
≤ / ≥ → ● pintada  |  < / > → ○ vazia

Regra direta: tem "traço embaixo" do sinal (≤ ≥)? Tem tinta na bolinha.

Álgebra e Produtos Notáveis

❌ Erro da morte Quadrado da Soma
(a + b)² = a² + b²

O erro mais clássico da álgebra. O termo do meio some.

✓ Correto
(a + b)² = a² + 2ab + b²

O 2ab nunca pode ser esquecido. Se der branco, abra a potência e faça (a+b)(a+b) no chuveirinho.

❌ Errado Distributiva
2(x + 5) = 2x + 5

O carteiro só entregou para o primeiro morador.

✓ Correto
2(x + 5) = 2x + 10

O número de fora multiplica todos os termos de dentro.

❌ Errado Distributiva com negativo
−3(x − 2) = −3x − 6

Esquecer o jogo de sinais no segundo termo.

✓ Correto
−3(x − 2) = −3x + 6

(−3) × (−2) = +6. Menos com menos dá mais.

❌ Crime matemático Fatoração em Frações
(x + 2) / x = 2  (cancelou o x)

Cortar termo que está somando, não multiplicando.

✓ Correto
Só cancela se estiver multiplicando (fatorado)

Fatore primeiro: (x+3)² / (x+3) = (x+3)(x+3) / (x+3) = x+3 ✓

❌ Errado Expoente na evidência
x⁵ − x² = x²(x³ − 0)

Achar que x² ÷ x² = 0.

✓ Correto
x⁵ − x² = x²(x³ − 1)

x² ÷ x² = 1, nunca 0. O 1 fica dentro do parênteses.

Relação de Ordem

❌ Errado Negativos
−8 > −2  (pois 8 > 2)

Confundir o módulo (valor absoluto) com o valor real.

✓ Correto
−8 < −2

Termômetro ou dinheiro: dever 8 reais é pior que dever 2. Mais à esquerda na reta = menor.

❌ Errado Multiplicação por negativo
3 > 2  →  ×(−1)  →  −3 > −2

Manter o sinal da desigualdade ao multiplicar por negativo.

✓ Correto
3 > 2  →  ×(−1)  →  −3 < −2

Multiplicou ou dividiu por número negativo? O sinal da desigualdade inverte.

Potenciação e Radiciação

❌ Errado Bases diferentes
2³ · 3² = 6⁵

Somar expoentes de bases diferentes.

✓ Correto
2³ · 3² = 8 · 9 = 72  (calcula separado)

Só une expoentes com a mesma base: 2³ · 2² = 2⁵.

❌ Errado Raiz de soma
√(a² + b²) = a + b

Distribuir a raiz sobre a soma.

✓ Correto
√(9 + 16) = √25 = 5  ≠ 3 + 4 = 7

A raiz distribui apenas sobre produto e quociente, nunca sobre soma ou subtração.
